# OpenClash Manager

A professional Home Assistant integration to manage your OpenClash configuration on OpenWrt routers.

## Key Features

- 🚀 **One-Tap Switching**: Change Clash configurations instantly.
- 🛡️ **Master Toggle**: Enable or disable OpenClash globally.
- 📡 **Mode Selection**: Switch between Rule, Global, and Direct modes.
- 🔄 **Service Actions**: Restart OpenClash and update subscriptions/cores from HA.
- 🍏 **Apple Home Optimized**: Custom metadata for a perfect HomeKit experience.
- 🔗 **Deep Linking**: Jump directly to the OpenClash Luci dashboard from your device.

## Prerequisites

- An OpenWrt router with OpenClash installed.
- SSH access enabled on the router.

## Setup

1. Add this repository to HACS.
2. Search for `OpenClash Manager` and install.
3. Restart Home Assistant.
4. Go to **Settings > Devices & Services > Add Integration** and select **OpenClash Manager**.
